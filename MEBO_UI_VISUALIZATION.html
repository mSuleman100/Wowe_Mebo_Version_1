Implementation Notes
Robot Type Selector: Located at the top of the control panel, allows switching between WOWE and MEBO modes. When WOWE is selected, the original Wowe control interface is displayed. When MEBO is selected, this Mebo interface is shown.

Color Scheme: Movement buttons use cyan (#27d3ff), action buttons use green (#32ffb7), and stop button uses red (#ff3b3b) to match the existing tactical theme.

Layout: The interface is organized into logical sections: Movement (D-pad style), Claw Gripper, Joint Rotation, Joint 1, and Joint 2. Each section is clearly labeled and grouped visually.
